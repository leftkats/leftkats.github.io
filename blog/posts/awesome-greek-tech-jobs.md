# awesome-greek-tech-jobs: Mapping Tech Hiring in Greece

Published: `2026-02-12T10:00:00Z`

This project is an open-source directory of companies hiring for tech roles in Greece.
It combines curated company data with practical hiring context so job seekers can search faster and smarter.

## What makes it useful

- Community-maintained list of hiring companies
- Coverage of remote, hybrid, and on-site work policies
- Sector-based exploration for focused job searches
- Daily-updated Workable snapshots to track market movement

## Why I like this format

Instead of fragmented links and scattered notes, everything is centralized in a transparent, versioned repository.
It is simple to contribute, easy to audit, and practical for both candidates and recruiters.
